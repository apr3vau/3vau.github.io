哲学的基本问题

思维和存在 何者是本源 的问题 （第一性问题）
  • 思维 => 唯心
  • 存在 => 唯物
思维能否 正确认识存在 的问题（同一性问题）
  • 同一 => 可知论
  • 不同一 => 不可知论

为何思维和存在的关系问题是哲学的基本问题

  • 是人们在生活和实践活动中遇到的和 无法回避的基本问题
  • 贯穿哲学的始终，是 一切哲学不能回避、必须回答 的问题
  • 决定着各种哲学的 基本性质和方向 ，决定着它们对其他哲学问题的回答
