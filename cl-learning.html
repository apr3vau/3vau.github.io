Common Lisp学习

基础语法

format
输出
list
创建列表
defvar
创建变量
setf
给变量赋值或创建变量（可以一次传入多对）
let
列表的一个元素创建局部变量，计算第二个元素的表达式
progn
计算内部所有表达式元素
car
取出列表第一个元素
cdr
取出列表除第一个外的所有元素
listp
判断是否为列表
numberp
判断是否为数字
if
第一个元素是条件，第二个元素是if计算内容，第三个元素是else计算内容
eql
判断两个元素是否相等
cons
把第一个元素加入第二个元素的列表中
remove
把第一个元素从列表（第二个列表）中移除
'
原封不动返回其元素
#'
后带函数，返回其函数本身
apply
返回把函数（第一个元素）作用到列表上的计算结果，第一个元素是函数本身，第二个元素是列表
read
读取并返回读取到的值
defun
定义函数
lambda
没有名字，不用单出来的函数
（概念）plist
给list里的每个项取一个名字 (list :a 1 :b 2)
getf
取出plist里的元素，第一个元素是plist，第二个元素是元素名
y-or-n-p
让用户输入y或n，一个字符串元素是提示内容
do
条件循环
loop
循环，通过(return)退出

format格式字符串

~a
占位符，把后面的一个元素输出出去
~t
调格式用，移动光标位置
~r
把数字转为英语读法
~v

可用于转义format格式符中的数字，效果如下： 

CL-USER> (format nil "~40:@<~A~>" "==================你好==================")
" ==================你好================== "
CL-USER> (format nil "~v:@<~A~>" 50 "==================你好==================")
"      ==================你好==================      "

文件名操作（pathspec/namestring/pathname）

概念
pathspec和namestring含义相同，都指的是一个字符串格式的路径，例如"~/Desktop/rosa" 而pathname则是lisp用于操作文件时使用的文件名格式，会在pathspec前加"#P"，如#P"~/Desktop/rosa"
(pathname <namestring>)
namestring转pathname
(namestring <pathname>)
pathname转namestring

quicklisp Common Lisp包管理器

加载包
(ql:quickload "包名")
卸载包（从本地）
(ql:uninstall "包名")
找包
(ql:system-apropos "包名")
使用某个包内的函数
(包名:函数)

osicat
